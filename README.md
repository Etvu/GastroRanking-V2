# GastroRanking-V2

Pour cette Application SPA GastroRanking les fonctionnalités sont : un CRUD pour les restaurant, le tri en fonction des pays/cuisine/nom/notes, mettre en favori ou non un restau, ajouter un avis, afficher le drapeau du pays et rechercher un restau avec son nom.

Durant ce projet j'ai utilisé 3 Design pattern:

Le MVC : avec le model qui gère les donnés et la logique, la vue qui affiche le dom et collecte les action de l'user et le controller qui fait le lien entre les 2 et ecoute les events.
Le module pattern pour découper le code et ne pas avoir de fichier trop long
L'observer pour que dès qu'on change les données l'interface change
